Which strategy should use to perform the non-intrusive integration for the compiling process.


The accepted entry is: {{artifact,{name of the artifact}} | {workspace,{relative path within the workspace}}

  • workspace: will refer the path to the workspace of the project specified
  • artifact: will refer to the artifact archived for the project

e.g.:

  • artifact,MODULE_INTROOT
  • workspace,MODULE_INTROOT
