Is where the dependency introot is located relative to the WORKSPACE or the artifact directory (build result type or job id), tipically the introot is named as, if MODULE-HEAD job as a dependency, as ALMASW-MODULE-HEAD/ACSSW, where MODULE-HEAD is a job to build MODULE by using almasw-modbuilder.

This field is optional, the default directory will be ALMASW-{DEPENDENCY JOB NAME}/ACSSW.
